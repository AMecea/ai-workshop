## Project Overview
 - This project aims to implement a MCP in Python for ...
 - This project has the following structure:
     - src/ dir stays the python modules
     - scripts/ dir should simple helper scripts


## When running this project
 - This project uses the python virtualenv initialized in .venv directory.
 - Install dependencies by creating the env `python3 -m venv .venv` and run `./venv/bin/pip install -r requirements.txt`
 - Run scripts with `.venv/bin/python script_file.py`
 - Run tests with `.venv/bin/pytest`
 - Prefer to run tests with –-disable-warnings and —-tb=short for less verbose output.
